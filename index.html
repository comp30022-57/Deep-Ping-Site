About

Deep Ping is a free Gmail plugin to ensure you never lose touch with friends, family, and colleagues.

Installation

Function

Deep Ping searches through your emails and displays a list of contacts who you haven't spoken to in a while. When it finds someone you want to reconnect with, you can click their name to view their last email or reply with a template. You can also use the ⊖ button to ignore them.

You should be able to find useful results in seconds with the default search options however not everyone uses Gmail the same way. If an initial search doesn't give you the results you're looking for, you can modify your search options or add custom filters to make sure you can always find what you need.

Convenience

Deep Ping is fully contained within Gmail so there's no need to use a new website or app.
